# 05 – Source Library

## Mire szolgál?

A projekt által használt források ellenőrzött bevételezésének, metaadatolásának, jogi státuszának és állításszintű visszakövethetőségének helye.

## Milyen fájlok kerülnek ide?

- Source Registerek és bibliográfiai rekordok;
- jog- és licencnyilvántartások;
- integritási hash-ek és bevételezési jegyzőkönyvek;
- jóváhagyott, jogszerűen tárolható forráspéldányok;
- forrás–állítás kapcsolati táblák.

## Mi nem kerülhet ide?

- ismeretlen eredetű fájl;
- engedély nélkül tárolt vagy terjesztett szerzői jogi anyag;
- generált szöveg forrásként feltüntetve;
- H5P-build vagy kész modul.

## Workflow-kapcsolat

Minden modul első kötelező állomása. Master Script csak `APPROVED` forrásrekordokra épülhet.

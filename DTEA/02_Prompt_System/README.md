# 02 – Prompt System

## Mire szolgál?

A mesterséges intelligenciával támogatott munkafázisok reprodukálható, verziózott és auditálható prompt-rendszerének helye.

## Milyen fájlok kerülnek ide?

- jóváhagyott system- és task-promptok;
- promptverziók és változásnaplók;
- bemeneti/kimeneti sémák és elfogadási kritériumok;
- tesztesetek, tiltólisták és prompt-értékelési jegyzőkönyvek;
- szerepkör- és workflow-specifikus promptcsomagok.

## Mi nem kerülhet ide?

- API-kulcs, személyes adat vagy nem publikálható forrástartalom;
- ellenőrizetlen, egyszeri chatmásolat mint hivatalos prompt;
- generált történelmi állítás forrásazonosító nélkül;
- kész tananyag vagy H5P-csomag.

## Workflow-kapcsolat

Elemzésnél, tervezésnél, asset-előkészítésnél és QA-támogatásnál használható. A promptkimenet mindig munkapéldány; szakmai jóváhagyás szükséges.

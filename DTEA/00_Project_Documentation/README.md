# 00 – Project Documentation

## Mire szolgál?

A projekt céljának, hatókörének, architektúrájának, szerepköreinek és végponttól végpontig működési modelljének elsődleges dokumentációs helye.

## Milyen fájlok kerülnek ide?

- projektkézikönyvek és architektúraleírások;
- teljes fejlesztési és modul-életciklus workflow-k;
- felelősségi mátrixok, döntési és változáskezelési elvek;
- onboarding- és működési útmutatók.

## Mi nem kerülhet ide?

- történelmi tananyag vagy forráspéldány;
- implementációs kód, H5P-csomag vagy médiaasset;
- modul-specifikus QA-bizonyíték;
- kiadási bináris.

## Workflow-kapcsolat

Ez a könyvtár határozza meg a teljes folyamatot, és minden új modul indításakor, kapuváltásakor és kiadásakor hivatkozási alapként szolgál.
